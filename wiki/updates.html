Wiki Updates - November 2017

Featured

CD Import Tool

As of version 8.13, the Import Tool will start checking all 3 email address columns for existing leads/contacts. Since that behavior is currently only available for the US data center, we do have legacy behavior documented on the Wiki page as well. As soon as EU and AU data centers have been updated, you shouldn't need the legacy behavior information because the change is transmitted as a cloud update.

Visit the CD Import Tool page

Additional Updates

  • The Event Connectors page has some clarification on what needs to be done when a connector is being re-added or re-authenticated.
  • We also got some clarification on what the ambiguous term "large Email Sends" means for the Profile Management Data Sync feature.
  • Survey questions can have the Type changed from the Survey Question record as documented here.
  • If your customer has a List Form Field and is trying to set the --Choose Option-- text as their default option, they can't do that without setting up an actual option in their picklist mapping. The Form Fields page goes into more detail on that.
  • When customers ask about Email Sends querying the recipients to send to and even specifically how the number on the confirmation screen is calculated, we explain how security roles can affect that number, on the Email Sends page. The Email Sends page has some other great processing information but this particular section was updated for further clarity.
