Additional Updates

  • The Plugin page now has information on troubleshooting the Plugin Registration tool.
  • The Subject field for Bulk Text Messages is explained here. One thing to keep in mind is that the Bulk Text Messages entity is different from the BulkSMS connector so when looking for information for either feature area, please be sure you are checking the correct page.
  • Email Sends sent by a Campaign Automation include some specific identifiers in the Correlation ID that can help you determine which Campaign Automation (and even which node in the Campaign Automation) generated the Send. This information has also been stored on the Send Email Action page.
  • If your customer sees a Soft Bounce message of 501 5.5.2 RCPT TO syntax error - parse exception occur, we detail what could cause that to happen and provide some troubleshooting guidance on it here.
  • We have clarification on which template will be used in a Split Test when the Copy Existing Template from Version A option is selected.
  • Permissions needed in order for a User to access the Email Statistics from the record's ribbon have also been clarified.
